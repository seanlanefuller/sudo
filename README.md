# Sudoku

A clean, minimalist Sudoku game built as a Progressive Web App (PWA). This project focuses on simplicity, responsiveness, and offline playability.

## Features

- **Clean UI**: A sleek, dark-themed interface designed for both desktop and mobile.
- **Progressive Web App**: Install it on your home screen for an app-like experience.
- **Offline Support**: Play anywhere, even without an internet connection, thanks to service workers.
- **Difficulty Levels**: Choose between Easy, Medium, and Hard.
- **Error Tracking**: Tracks mistakes (up to 3) to keep the challenge fair.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Modern layout and styling.
- **JavaScript**: Game logic and board generation.
- **Service Workers**: For offline capabilities.
- **Manifest**: PWA configuration.

## How to Run

Simply open `index.html` in your favorite web browser, or serve it using a local development server like `npx serve .`.
